**promise面试问题一：**
```
支持链式调用，可以解决回调地狱
	1.什么是回调地狱
	回调函数的嵌套调用，外部函数的异步执行的结果是嵌套的回调执行的条件
	
	2.回调地狱的缺点
	不便于阅读，和异常处理
	
	3.解决方案
	promise链式调用
		
```
